== Description ==
Introducing the ultimate lightweight Google Tag Manager plugin for WordPress!
We\'ve stripped away all unnecessary features and code, resulting in a plugin that\'s lightning fast and ultra-lightweight. 
With our streamlined interface, you can easily add your Google tag without any coding knowledge, in just a few clicks.

**Features:**

* Simple and guided setup
* Lightweight and fast

**Included languages:**

* English

== Installation ==
1. Download the plugin and unzip it.
2. Upload the folder liayntech-lightag/ to your /wp-content/plugins/ folder.
3. Activate the plugin from your WordPress admin panel.
4. Go to settings page, and save your Google Tag Manager container.
5. Done.
